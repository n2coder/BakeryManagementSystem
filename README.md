# 🍞 Bakery Sales Analysis Dashboard

## 📊 Overview
This project analyzes bakery sales data to extract business insights such as top-selling items, revenue trends, and daily performance.

## 🚀 Features
- KPI Metrics (Revenue, Orders, Top Item)
- Sales Trend Analysis
- Item Performance Visualization

## 🛠 Tech Stack
- Python
- Pandas
- Streamlit
- Matplotlib

## 📌 Key Insights
- Cake generates highest revenue
- Weekend sales are higher
- Some items are slow-moving
